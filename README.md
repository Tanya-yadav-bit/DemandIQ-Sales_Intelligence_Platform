<h1 align="center"> Sales Prediction Dashboard</h1>

<p align="center">
A Machine Learning powered interactive dashboard built with Streamlit to analyze and predict sales.
</p>

<hr>

<h2>🚀 Live Demo</h2>
<p>
👉 (Add your Streamlit link here after deployment)
</p>

<hr>

<h2>📌 Features</h2>
<ul>
  <li>📊 Interactive data visualization using Plotly</li>
  <li>🤖 Machine Learning prediction (XGBoost model)</li>
  <li>📈 PCA & KMeans clustering insights</li>
  <li>📂 CSV file upload for predictions</li>
  <li>⚡ Fast and user-friendly Streamlit UI</li>
</ul>

<hr>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li>Python</li>
  <li>Streamlit</li>
  <li>Pandas & NumPy</li>
  <li>Scikit-learn</li>
  <li>XGBoost</li>
  <li>Plotly</li>
</ul>

<hr>

<h2>📁 Project Structure</h2>

<pre>
app.py
model.pkl
kmeans.pkl

clean_data.csv
requirements.txt
</pre>

<hr>

<h2>⚙️ Installation</h2>

<pre>
git clone https://github.com/Tanya-yadav-bit/DemandIQ-Sales_Intelligence_Platform
cd DemandIQ-Sales_Intelligence_Platform
pip install -r requirements.txt
streamlit run app.py
</pre>

<hr>

<h2>📊 How It Works</h2>
<ul>
  <li>Data is preprocessed and scaled</li>
  <li>PCA is applied for dimensionality reduction</li>
  <li>KMeans is used for clustering</li>
  <li>XGBoost model predicts sales</li>
</ul>
